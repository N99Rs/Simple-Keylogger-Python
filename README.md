# Voici un simple keylogger en python
Vous devez avoir python installé sur votre pc pour faire fonctionner le programme

# Comment ça marche ?
Ouvrez un invite de commande et saisissez "**pip install pynput**"
Vous devez accéder au dossier et éxécuter le fichier python.

Les frappes sont **enregistrées** dans un fichier texte dans ce dossier.

**Fermez le programme pour arrêter l'enregistrement de la saisie**
